# KioskRaspberry-Chromium
Kiosk mode for raspberry pi chromium setup. Setup for raspberry pi os, lite being the best for this, and it will configure an optimized chromium with all things needed to run non stop kiosk.

```
bash <(curl -s https://raw.githubusercontent.com/IRENNIE74/KioskRaspberry-Chromium/main/kiosk_setup.sh)
```
